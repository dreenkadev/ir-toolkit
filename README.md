# Ir Toolkit

Incident response data collection toolkit.

Toolkit pengumpulan data respons insiden.


## Description

Automates collection of forensic artifacts during incident response including memory, disk, and network information.

## Deskripsi

Mengotomatisasi pengumpulan artefak forensik selama respons insiden termasuk informasi memori, disk, dan jaringan.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Collect: sudo python ir_collect.py
- Output dir: python ir_collect.py --output /evidence
- Demo: python ir_collect.py --demo

## Penggunaan

- Kumpulkan: sudo python ir_collect.py
- Dir output: python ir_collect.py --output /evidence
- Demo: python ir_collect.py --demo
